Options
When Enabled, all the values of a cookie are visible in the interface.
When Disabled, only the name and the value of the cookie are visible in the interface.
When Enabled, Cookie-Editor will be added to the DevTools tabs. Devtools needs to be restarted to take effect when turning this option off.
Selects the behavior of the export button. When "Ask every time" is selected, a menu asking you to chose the format will show up when you press the export button.
When selecting "Auto", Cookie-Editor will match your browser's theme automatically.
